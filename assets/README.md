# 設計資產（待匯入）

請向活動／設計窗口取得並置於此資料夾（建議子資料夾：`key_visual/`、`logo/`、`cis/`、`patterns/`）。

| 項目 | 說明 |
|------|------|
| Key Visual | 活動主視覺 |
| Logo | 公司／活動標誌（向量與匯出規格依 UI 需求） |
| 印花圖樣 | 背景或裝飾用圖樣 |
| CIS | 企業識別系統規範（色彩、字體、使用準則等） |

檔名建議含版本與日期，避免覆蓋。

**線框圖（Wireframe）**目前集中於 `docs/design/wireframe/`（見 `docs/README.md`）。
